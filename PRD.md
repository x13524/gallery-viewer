# 相册浏览器 PRD

> 版本 v1.0 | 2026-05-09 | 状态：已实现

***

## 一、产品定位

浏览器本地运行的相册浏览工具，使用 File System Access API 打开本地文件夹，快速浏览图片和视频，无需服务器、不上传文件。提供专业的暗色主题、高性能缩略图缓存、自适应网格布局，支持大型目录浏览。

***

## 二、目标用户

- 自己使用（个人工具）
- 场景：浏览本地截图、素材、照片、视频

***

## 三、与技术模板的差异

对比 Gallery-Viewer-HTML，本方案做了以下取舍和增强：

| 功能 | Gallery-Viewer-HTML | 本方案 |
|---|---|---|
| 文件树侧边栏 | 虚拟滚动 | 普通 DOM，不做虚拟滚动 |
| 侧边栏平铺模式 | 无 | 树形 + 全平铺双模式 |
| 面包屑导航 | 无 | 有，点击跳转任意上级目录 |
| IndexedDB 缩略图缓存 | 有 | 保留，新增大小限制（256MB）和自动清理机制 |
| IndexedDB 文件夹恢复 | 无 | 有，刷新页面自动恢复上次文件夹和最后选中目录 |
| 拖拽打开文件夹 | 无 | 暂未实现 |
| 文件管理 | 删除/重命名/移动，Ctrl+Z | 去掉，只读浏览 |
| 全屏查看 | 图片 + 视频 + 音频 | 图片 + 视频，去掉音频 |
| 视频自动播放 | 无 | 弹窗打开即播，侧边视频默认静音暂停 |
| 弹窗缩放 | 图片左上角为原点 | 以鼠标位置为中心 |
| 小图处理 | 被 contain 放大 | 限制原始尺寸不放大，避免模糊 |
| 相邻图片预加载 | 无 | 有，左右切换即时 |
| EXIF 元数据 | GPS/地图/视频详情/音频 ID3 | 简化，有数据才弹，无则不弹 |
| 文件详情面板 | 无 | 右侧可折叠，单击查看/双击弹窗 |
| 缩略图 | 无统一比例 | 4:3 统一容器，contain 不裁切 |
| 排序优化 | 每次加载文件元数据 | 默认按名称不调 getFile()，切换排序才加载，新增按类型排序 |
| 加载指示器 | 无 | 有，延迟显示避免闪烁 |
| 鼠标悬停边缘滚动 | 有 | 去掉 |
| 右键菜单 | 有 | 去掉 |
| 音频播放 + ID3 | 有 | 去掉 |
| 外部依赖 | exif-js, SparkMD5, FontAwesome | exif.js（本地）+ xgplayer（本地），无外部 CDN |
| 架构 | 单文件内联 | HTML + 外部 CSS + 外部 JS |
| 网格列数控制 | 手动输入 | 自适应，根据缩略图尺寸和窗口宽度自动计算 |
| 缩略图尺寸控制 | 滑块（120-500px） | 预设三档（小/中/大） |
| 媒体类型筛选 | 无 | 有（全部/图片/视频） |
| 子目录包含开关 | 无 | 有，可选择是否包含子目录媒体 |
| 刷新当前目录 | 无 | 有，手动刷新获取文件系统变更 |
| 偏好持久化 | 无 | localStorage 保存用户偏好（视图模式、排序、缩略图尺寸、筛选等） |
| 大目录优化 | 无 | 有，≥1200 个文件自动进入大目录模式，分批渲染 |
| 视频播放器 | 原生 | 弹窗使用 xgplayer，侧边使用原生 video |

## 四、功能列表

### 已实现

- [x] 打开本地文件夹
  - File System Access API（`showDirectoryPicker`）
  - 刷新页面自动恢复上次文件夹（IndexedDB 持久化 handle + queryPermission）
  - 自动恢复上次最后选中的子目录
- [x] 递归遍历子文件夹，只显示含媒体文件的文件夹
- [x] 侧边栏双模式，可一键切换：
  - **树形视图**：层级缩进展示所有包含媒体文件的文件夹
  - **平铺视图**：全部文件夹平铺为单层列表
- [x] 侧边栏每个文件夹后显示总媒体文件数（含子目录）
- [x] 面包屑导航，路径任意层级可点击跳转
- [x] 支持的格式
  - 图片：JPG、JPEG、PNG、GIF、WebP、BMP、SVG、AVIF、HEIC
  - 视频：MP4、WebM、MOV、MKV、AVI、M4V
- [x] 自适应网格视图
  - 4:3 统一容器比例 + contain，完整显示不裁切
  - 缩略图用 Canvas 生成，视频截取首帧
  - 文件名显示在缩略图下方，卡片显示尺寸/时长和文件大小
  - 自动计算列数，根据窗口宽度和缩略图尺寸自适应
  - 缩略图懒加载（IntersectionObserver）
  - 分批渲染（每批 120 个），大目录不卡顿
- [x] IndexedDB 缩略图缓存
  - Key：文件路径 + 缩略图预设 + 版本
  - 上限 256MB，超过自动清理最早的缓存
  - 接近上限（224MB）提示用户
  - 提供手动清除缓存按钮
- [x] 大图/视频查看（Modal）
  - 图片：鼠标滚轮缩放（以鼠标位置为中心）、拖拽平移
  - 小图以原始尺寸显示，不超过屏幕 90%，避免放大模糊
  - 视频：使用 xgplayer 播放器，自动播放有声，支持倍速、画中画
  - 键盘 ← → 切换，Esc 关闭（capture 阶段监听），背景点击关闭
  - 预加载相邻图片，切换无延迟
- [x] 文件名搜索（实时过滤，防抖 240ms）
- [x] 排序：文件名 / 修改日期 / 文件大小 / 文件类型，升降序
  - 默认按名称排序，不加载文件元数据
  - 切换为大小/日期排序时按需调用 getFile()
- [x] 缩略图尺寸预设（小/中/大）
  - 小：160px 缩略图，150px 容器高度
  - 中：220px 缩略图，210px 容器高度（默认）
  - 大：320px 缩略图，280px 容器高度
- [x] 媒体类型筛选（全部/图片/视频）
- [x] "包含子目录"开关，可选择是否显示子目录媒体
- [x] 刷新当前目录按钮，手动获取文件系统变更
- [x] 右侧文件详情面板（可折叠，默认展开）
  - 图片：直接显示原始文件 Object URL，确保清晰度
  - 视频：原生 video 标签，默认静音暂停
  - 显示文件名、路径、格式、大小、修改时间、分辨率
  - 视频额外显示时长
  - 有 EXIF 时显示 EXIF 信息
  - 面板展开时：单击查看详情，双击弹窗预览
  - 面板收起时：单击直接弹窗
- [x] EXIF 信息（有数据才显示）
  - 拍摄时间、设备、光圈、快门、ISO、焦距
- [x] 加载指示器（延迟 360ms 显示，避免快速操作闪烁）
- [x] 暗色主题（全程暗色，专业视觉）
- [x] 清除缓存
- [x] 偏好持久化（localStorage）
  - 侧边栏视图模式
  - 排序方式和方向
  - 缩略图尺寸预设
  - 详情面板展开/收起
  - 媒体类型筛选
  - 是否包含子目录
  - 最后选中的文件夹路径
- [x] 大目录模式（≥1200 个文件）
  - 自动检测并提示用户
  - 分批渲染 + 懒加载确保流畅

### 不做的事

- 文件管理（删除、重命名、拖拽移动）
- 音频播放
- 云端上传
- 文件树虚拟滚动（个人文件夹不过几千个）
- 地图跳转（GPS 坐标）
- 右键菜单
- 鼠标悬停边缘滚动

## 五、技术方案

| 层 | 选型 |
|---|---|
| 架构 | 单 HTML 入口 + 外部 CSS + 外部 JS |
| 文件系统 | File System Access API（handle 持久化到 IndexedDB） |
| 缩略图缓存 | IndexedDB，256MB 上限，LRU 自动清理 |
| 图片缩放 | CSS transform + 以鼠标位置为中心计算偏移 |
| 视频首帧 | Canvas drawImage(video) |
| 视频播放器 | 弹窗：xgplayer；侧边：原生 video |
| EXIF | exif.js（本地引入） |
| 外部依赖 | exif.js + xgplayer，均本地引入，无 CDN |
| 自适应网格 | CSS Grid + `auto-fit` + `minmax()` |
| 懒加载 | IntersectionObserver，提前 280px 预加载 |
| 分批渲染 | requestAnimationFrame 分批插入 DOM |
| 防抖 | 搜索 240ms，缓存审计延迟执行 |
| 持久化 | IndexedDB（文件夹 handle + 缩略图）+ localStorage（用户偏好） |

## 七、关键设计决策

### 7.1 缩略图策略
- 小图保持原始尺寸，不放大避免模糊
- PNG/GIF/BMP/SVG 保持无损，转 PNG 缓存
- JPG/AVIF/HEIC 转 JPEG 0.9 质量缓存
- 视频首帧转 JPEG 0.85 质量缓存

### 7.2 性能策略
- 默认按文件名排序，不提前读取文件元数据
- 切换目录或排序时才按需批量读取元数据
- 缩略图懒加载，仅渲染可见区域附近
- 大目录分批渲染，避免长时间阻塞主线程
- 使用 renderToken 防止竞态条件和旧数据回写

### 7.3 缓存策略
- 256MB 硬上限，超过自动清理最早的缓存
- 224MB 软警告，提示用户
- 每次访问缓存更新时间戳，实现 LRU 效果
- 切换缩略图尺寸时重新生成不同尺寸的缓存

### 7.4 视频策略
- 侧边详情：原生 video，默认静音暂停，轻量简单
- 弹窗预览：xgplayer，功能完整，体验更佳
- 打开弹窗时自动暂停侧边视频
- 关闭弹窗后不自动恢复侧边视频

### 7.5 交互策略
- Esc 键关闭弹窗监听在 capture 阶段，避免被播放器拦截
- 详情面板收起/展开无动画，避免文案挤压和抖动
- 全局字体使用系统原生字体栈，避免加载闪烁
- 切换文件夹自动选中第一张图，避免空状态
