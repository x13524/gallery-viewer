# 相册浏览器 PRD

> 版本 v0.2 | 2026-05-07 | 状态：已实现

***

## 一、产品定位

浏览器本地运行的相册浏览工具，用 File System Access API 打开本地文件夹，快速浏览图片和视频，无需服务器、不上传文件。

***

## 二、目标用户

- 自己使用（个人工具）
- 场景：浏览本地截图、素材、照片

***

## 三、与技术模板的差异

对比 Gallery-Viewer-HTML，本方案做了以下取舍和增强：

| 功能 | Gallery-Viewer-HTML | 本方案 |
|---|---|---|
| 文件树侧边栏 | 虚拟滚动 | 普通 DOM，不做虚拟滚动 |
| 侧边栏文件数量 | 无 | 有 |
| 侧边栏平铺模式 | 无（仅树形） | 树形 + 全平铺双模式 |
| 面包屑导航 | 无 | 有，点击跳转任意上级目录 |
| IndexedDB 缩略图缓存 | 有 | 保留 |
| IndexedDB 文件夹恢复 | 无 | 有，刷新页面自动恢复 |
| 拖拽打开文件夹 | 无 | 有 |
| 文件管理 | 删除/重命名/移动，Ctrl+Z | 去掉，只读浏览 |
| 全屏查看 | 图片 + 视频 + 音频 | 图片 + 视频，去掉音频 |
| 视频自动播放 | 无 | 弹窗打开即播 |
| 弹窗缩放 | 图片左上角为原点 | 以鼠标位置为中心 |
| 小图处理 | 被 contain 放大 | 限制原始尺寸不放大 |
| 相邻图片预加载 | 无 | 有，左右切换即时 |
| EXIF 元数据 | GPS/地图/视频详情/音频 ID3 | 简化，有数据才弹，无则不弹 |
| 文件详情面板 | 无 | 右侧可折叠，单击查看/双击弹窗 |
| 缩略图 | 无统一比例 | 4:3 统一容器，contain 不裁切 |
| 排序优化 | 每次加载文件元数据 | 默认按名称不调 getFile()，切换排序才加载 |
| 加载指示器 | 无 | 有 loading spinner |
| 鼠标悬停边缘滚动 | 有 | 去掉 |
| 右键菜单 | 有 | 去掉 |
| 音频播放 + ID3 | 有 | 去掉 |
| 外部依赖 | exif-js, SparkMD5, FontAwesome | 仅 exif-js（CDN 可选） |
| 架构 | 单文件内联 | HTML + 外部 CSS + 外部 JS |

## 四、功能列表

### 已实现

- [x] 打开本地文件夹
  - File System Access API（`showDirectoryPicker`）
  - 支持拖拽文件夹到页面直接打开
  - 刷新页面自动恢复上次文件夹（IndexedDB 持久化 handle + queryPermission）
- [x] 递归遍历子文件夹，只显示含媒体文件的文件夹
- [x] 侧边栏双模式，可一键切换：
  - **树形视图**：层级缩进展示所有包含媒体文件的文件夹
  - **平铺视图**：全部文件夹平铺为单层列表
- [x] 侧边栏每个文件夹后显示文件数量
- [x] 面包屑导航，路径任意层级可点击跳转
- [x] 支持的格式
  - 图片：JPG、JPEG、PNG、GIF、WebP、BMP、SVG
  - 视频：MP4、WebM、MOV、MKV、AVI
- [x] 网格视图
  - 4:3 统一容器比例 + contain，完整显示不裁切
  - 缩略图用 Canvas 生成，视频截取首帧
  - 文件名显示在缩略图下方
- [x] IndexedDB 缩略图缓存
  - Key：文件路径 + 缩略图尺寸
  - 二次打开同一文件夹秒加载
  - 提供清除缓存按钮
- [x] 大图/视频查看（Modal）
  - 图片：鼠标滚轮缩放（以鼠标位置为中心）、拖拽平移
  - 小图以原始尺寸显示，不超过屏幕 90%
  - 视频：自动静音播放，支持原生控件
  - 键盘 ← → 切换，Esc 关闭，背景点击关闭
  - 预加载相邻图片，切换无延迟
- [x] 文件名搜索（实时过滤）
- [x] 排序：文件名 / 修改日期 / 文件大小，升降序
  - 默认按文件名排序，不加载文件元数据
  - 切换为大小/日期排序时按需调用 getFile()
- [x] 缩略图尺寸滑块（120-500px）
- [x] 网格列数控制（2-10 列）
- [x] 右侧文件详情面板（可折叠，默认展开）
  - 预览图 + 文件名、类型、大小、修改日期
  - 图片：尺寸、MIME 类型
  - 视频：分辨率、时长
  - 面板展开时：单击查看详情，双击弹窗查看
  - 面板收起时：单击直接弹窗
- [x] EXIF 信息（有数据才弹窗）
  - 拍摄时间、相机型号、分辨率
  - 光圈、ISO、焦距、快门速度
- [x] 加载指示器
- [x] 暗色主题（全程暗色）
- [x] 清除缓存

### 不做的事

- 文件管理（删除、重命名、拖拽移动）
- 音频播放
- 云端上传
- 文件树虚拟滚动（个人文件夹不过几百个）
- 地图跳转（GPS 坐标）
- 右键菜单
- 鼠标悬停边缘滚动

***

## 五、技术方案

| 层 | 选型 |
|---|---|
| 架构 | 单 HTML 入口 + 外部 CSS + 外部 JS |
| 文件系统 | File System Access API（handle 持久化到 IndexedDB） |
| 缩略图缓存 | IndexedDB |
| 图片缩放 | CSS transform + 以鼠标位置为中心计算偏移 |
| 视频首帧 | Canvas drawImage(video) |
| EXIF | exif-js（CDN 按需加载，未加载或无数据不弹窗） |
| 外部依赖 | 核心零依赖，exif-js 可选 |
