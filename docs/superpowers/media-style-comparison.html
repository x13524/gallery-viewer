A 适合什么情况？

你主要浏览照片、截图、轻量素材，希望界面尽可能安静，顶部和详情都弱化。

B 适合什么情况？

你更重视沉浸感和媒体封面，希望一打开就像视频/图片媒体库，缩略图更有存在感。

C 适合什么情况？

你经常在项目资源目录里快速筛素材，希望目录、列表、详情同时高效可见。
